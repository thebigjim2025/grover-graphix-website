# Grover Graphix — Corporate Website

The central hub for [Grover Graphix](https://grovergraphix.com): an animated, 3D single-page site showcasing our capabilities and linking to the live solutions we've built.

## Highlights

- Static site — no build step (`index.html` + `css/` + `js/`; Three.js r128 via jsdelivr CDN)
- Scroll-driven 3D camera journey through five scene stations
- Particle wordmark, bloom glow, floating device mockups, ring tunnel, DNA helix, wireframe grid floor
- Adaptive quality: sheds decoration automatically on low-powered devices
- Responsive, mobile-friendly, reveal animations, animated counters

## Live solutions featured

- [Prenota](https://prenota.com.au) — bookings that book themselves
- [Empowered Tradie](https://empoweredtradie.com.au) — your trade, empowered
- [Prenota Websites](https://rebuild.prenota.com.au) — booking sites, built free

## Deploy

Any static host works. Currently deployed on Cloudflare at grovergraphix.com.

```bash
# local preview
python3 -m http.server 8080
```
